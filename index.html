Internally, computers store dates as numbers. Various systems rely on different implementations. In JavaScript (implemented across web browsers) the date is stored as a number of milliseconds since midnight January 1, 1970 UTC. The conversion is very simple. The date is obtained through a constructor: date = new Date(milliseconds); while the number is obtained by: milliseconds = Date.parse(date); without the need for the 1 year = 365 * 24 * 60 * 60 * 1000 milliseconds arithmetics and leap year considerations. This can be exploited if you want to know e.g. how many seconds old you are! Just compare the given date with now.

How to use the calculator: you may set any date (Date 1, Date 2) or time difference in seconds you like. Obviously, by changing one of the three values, the equality is no longer satisfied. Then you may click the ↻ button to recalculate the corresponding value so that the equality holds, keeping the other two given parameter values.

Always think twice before entering your personal data. This site doesn't collect any data at all (you can trust me or inspect the source code), but there are lots of seemingly fu" web apps out there that do.
